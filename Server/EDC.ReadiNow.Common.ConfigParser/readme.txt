~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
 EDC.ReadiNow.Common.ConfigParser
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

The purpose of this assembly is to provide:
1. Classes to represent the raw entity/data content found in XML config files.
2. Parser to read the XML and produce these objects.
3. Basic mechanism for mapping aliases to entities, without reference to a database.
4. Basic mechanism for querying entity schema information from parsed entities (i.e. inheritance info, relationships, etc)
